## 실습 단계

<Steps>
1. **Node.js 설치 확인**

   ```bash
   node --version  # v20.x.x 이상
   npm --version   # 10.x.x 이상
   ```

   Node.js가 없다면 [nodejs.org](https://nodejs.org)에서 설치.

2. **Claude Code 설치**

   ```bash
   npm install -g @anthropic-ai/claude-code
   claude --version
   ```

3. **Anthropic API 키 설정**

   [console.anthropic.com](https://console.anthropic.com)에서 API 키 발급 후:

   ```bash
   # ~/.bashrc 또는 ~/.zshrc에 추가
   export ANTHROPIC_API_KEY="sk-ant-..."

   # 적용
   source ~/.bashrc
   ```

4. **Claude Code 첫 실행**

   ```bash
   mkdir ~/hello-agent && cd ~/hello-agent
   claude "Python으로 'Hello, AI Engineering 2026!' 출력하는 파일을 만들어줘."
   python hello.py
   ```

5. **DGX 서버 접속**

   <Tabs>
     <TabItem label="SSH">
       ```bash
       ssh [학번]@dgx.chu.ac.kr
       # 초기 비밀번호: 학번
       passwd  # 즉시 변경 필수!
       ```
     </TabItem>
     <TabItem label="SSH Config">
       ```
       # ~/.ssh/config
       Host dgx
         HostName dgx.chu.ac.kr
         User [학번]
         IdentityFile ~/.ssh/id_ed25519
       ```
     </TabItem>
   </Tabs>

6. **저장소 Fork 및 클론**

   ```bash
   # GitHub에서 Fork 후
   git clone https://github.com/[YOUR_USERNAME]/ai-engineering-2026.git
   cd ai-engineering-2026
   npm install
   npm run dev  # 로컬 사이트 확인
   ```
</Steps>
