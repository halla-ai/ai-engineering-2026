## Common Feedback

<Aside type="caution" title="Submission Guidelines">
1. **Exclude `.pyc` files** — Add `__pycache__/` and `*.pyc` to `.gitignore`
2. **Only modify your own student ID folder** — Don't modify other students' files or course materials when working from a fork
3. **No Korean filenames** — macOS NFD decomposition issues. Use English filenames
4. **Don't put assignment files in `src/content/docs/`** — This is the course site source directory
5. **Check before PR** — Review changed file list with `git diff --stat origin/main`
</Aside>
